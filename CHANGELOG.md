## [0.1.0] - 2026-01-04

### Added
- Initial release of Athesa framework
- Protocol-oriented design with ProcessProtocol, StateProtocol, HandlerProtocol, BridgeProtocol
- State machine engine for workflow management
- Screen detection system with multiple strategies
- Action executor for browser operations
- Command Pattern for action handling
- Strategy Pattern for screen detection
- Event system (pure Python, no dependencies)
- Process factory registry
- Selenium adapter (SeleniumBridge)
- Comprehensive documentation
- Unit test suite (47 tests)
